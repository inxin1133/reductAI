# 관리자 GNB (예상 구성안)

아래 메뉴는 현재 관리자 페이지 구조를 기준으로, 새로 추가된 스키마(LLM 사용 로그/가격/결제/크레딧)를 반영해 확장한 구성입니다.  
각 항목의 설명은 운영·관리 관점에서 필요한 기능을 간단히 요약합니다.

## 일반
- 대시보드  
  서비스 전반의 핵심 지표(사용량, 매출, 크레딧 소진, 에러율)를 한눈에 확인합니다.

## 관리
- 회원 및 테넌트
  - 회원 관리  
    사용자 계정 상태, 인증 상태, 접근 권한을 관리합니다.
  - 테넌트 관리  
    테넌트 기본 정보, 상태, 멤버 수/제한 등 멀티테넌트 운영을 관리합니다.
  - 역할 및 권한 관리  
    RBAC 역할/권한 정의 및 사용자·테넌트 권한 매핑을 관리합니다.

- AI 서비스
  - AI 제공업체(Providers)  
    제공업체 기본 정보와 상태를 관리합니다.
  - AI API Key(Credentials)  
    테넌트별 API 키/자격증명 정보를 관리합니다.
  - AI 모델 관리  
    모델 메타데이터, 상태, 노출 여부를 관리합니다.
  - 모델 접근 권한  
    테넌트별 모델 접근/우선순위/제한을 설정합니다.
  - 프롬프트 템플릿  
    기능/목적별 프롬프트 템플릿을 관리합니다.
  - 예시 프롬프트(Prompt Suggestions)  
    UI에 노출되는 추천 프롬프트를 관리합니다.
  - 출력 계약(Response Schemas)  
    모델 응답 스키마(계약)를 정의/관리합니다.
  - 모델 API 프로필(Model API Profiles)  
    Provider/모달리티별 호출·응답 매핑 프로필을 관리합니다.
  - Provider 인증 프로필(Auth Profiles)  
    인증 방식/스코프/토큰 발급 설정을 관리합니다.
  - 모델 라우팅 규칙  
    조건 기반 모델 선택/폴백 규칙을 운영합니다.
  - 모델 사용 로그  
    `schema_llm_usage.sql` 기반 공통 로그와 모달리티별 상세 사용량을 조회합니다.
  - 웹검색 정책  
    웹 검색 허용/제한/예산 정책을 관리합니다.

- 가격/요율 관리 (Pricing)  
  `schema_pricing.sql` 기반 가격표, SKU, Rate Card, 마진 정책을 관리합니다.
  - Rate Card/버전 관리  
    유효일 기준 요율 버전과 변경 이력을 관리합니다.
  - 모델/모달리티 요율표  
    입력/출력/캐시/이미지/영상/오디오/검색 요금을 설정합니다.
  - 마진 정책  
    모달리티/모델별 마진율 정책을 운영합니다.
  - 사용자 공개 요금표  
    사용자 안내용 평균 요금(1K 토큰 기준)을 산출/노출합니다.

- 크레딧/포인트 (Credits)  
  `schema_credits.sql` 기반 크레딧 계정, 원장, 그랜트, 사용 분배를 관리합니다.
  - 크레딧 계정/풀  
    테넌트 풀과 사용자 할당 계정을 조회/관리합니다.
  - 크레딧 원장(ledger)  
    발행/사용/조정/만료 등 모든 변동 이력을 추적합니다.
  - 그랜트/분배 정책  
    테넌트→사용자 크레딧 분배 및 회수 정책을 관리합니다.
  - 충전 상품 관리  
    크레딧 충전 상품(Top-up)과 가격/수량을 관리합니다.
  - 크레딧 사용 분배  
    사용 이벤트별 계정 분배 내역을 확인합니다.

- 결제 및 구독 (Billing)  
  `schema_billing.sql` 기반 플랜/구독/인보이스/결제를 관리합니다.
  - 구독 플랜 관리  
    요금제, 좌석/스토리지 한도, 추가 인원 비용을 관리합니다.
  - 플랜 가격/버전  
    월/연 가격 및 버전 변경 이력을 관리합니다.
  - 구독 현황  
    테넌트별 구독 상태, 기간, 업/다운그레이드를 관리합니다.
  - 청구서(Invoices)  
    청구 내역, 세금(VAT), 환율 적용 금액을 관리합니다.
  - 결제 내역(Transactions)  
    결제/환불/조정 거래를 확인합니다.
  - 결제 수단/PG 설정  
    결제 수단과 PG(토스/Stripe) 설정을 관리합니다.
  - 세금/환율 관리  
    VAT/운영 환율 정책을 설정합니다.

## 설정
- 시스템 설정
  - 서비스(Services) 관리  
    마이크로서비스 정의 및 상태를 관리합니다.
  - 보안 정책  
    접근 제어, 보안 정책, 인증 관련 설정을 관리합니다.
  - 감사 로그(Audit)  
    관리자 작업 이력을 추적/감사합니다.

- 다국어(i18n)
  - 지원 언어 관리  
    서비스에서 지원하는 언어 목록을 관리합니다.
  - 네임스페이스 관리  
    번역 키 네임스페이스를 관리합니다.
  - 번역 데이터 관리  
    번역 키/문구를 등록·수정합니다.
  - 번역 이력  
    번역 변경 이력을 확인합니다.
