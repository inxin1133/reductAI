### 서비스 결제 정책
- 월 선불 결제 환불 불가능, 구독 취소 시 이번 달(한달 채움)까지 기능 유지 후 유료 구독 중지(예약 취소)
- 연 선불 결제 환불
    - 월 정상가 적용
    - $200결제, 1/1 시작, 5/10 해지 시(5/31까지 사용 가능), 1~5개월 = 5개월 사용, $20(정가)x5개월 = $100
    - 최종 환불금: $100 = $200(결제금) - $100(차감분)
- 등급 변경 규칙
    - 월 결제 업그레이드: 즉시 반영 + 남은 기간만큼 “가격 차액” 비례 청구 + 남은 기간만큼 “크레딧 차이” 비례 추가 지급
    - 연 결제 업그레이드: 즉시 반영 + (남은 날짜 만큼 “가격 차액” 비례 청구 + 남은 날짜 만큼 “크레딧 차이” 비례 추가 지급) + 남은 월 “가격 차액” 비례 청구
    - 월 다운그레이드: 즉시 반영하지 않고, “다음 결제일부터 적용”(예약 변경)
    - 연 결제 다운그레이드: 환불 처리하고, 원하는 서비스 등급 결제 진행
