# 서비스 등급 자동 결제 및 환불정책

## 정책사항 기준
- 업그레이드는 즉시 적용
- 다운그레이드는 다음 결제일부터 적용
- 연간은 약정 할인
- 연간 약정 중도 취소 시 → 할인이 적용되지 않는 비용으로 계산 후, 남은 차액만 환불

### 월 선불 결제 구독취소
- 구독 취소 페이지로 이동 (환불 결제 없음)
- 구독 취소 시 이번 달(한달 채움)까지 기능 유지 후 유료 구독 중지 진행
- 예약 취소

### 연 선불 결제 구독취소 및 환불
- 구독 취소 페이지로 이동 (환불 결제 있음)
- 구독 취소 시 월 정상가 적용
- $200결제, 1/1 시작, 5/10 해지 시(5/31까지 사용 가능)
- 해당 월 구독에 대해서는 예약 취소
- 1~5개월 = 5개월 사용, $20(정가)x5개월 = $100
- 최종 환불금: $100 = $200(결제금) - $100(차감분)
- 환불 처리

### 월간, 서비스 등급 업그레이드
- 결제 페이지로 이동 (차액 결제 발생)
- 월간 서비스 등급 업그레이드
- 즉시 반영
- 월간 요금제 및 월 제공 크레딧 비례 정산
- 다음 자동 결제
[예상 시나리오]
- 현재 요금제: Pro
    - $20 / 월
    - 20,000 크레딧
- 변경 요금제: Premium
    - $60 / 월
    - 60,000 크레딧
- 결제 주기 시작일: 매월 10일
- 오늘 날짜: 2월 26일
(현재 사용 구간: 2월 10일 ~ 3월 9일, 남은 기간: 2월 26일 ~ 3월 9일(남은 일 : 12일)
1. 남은 일수 계산
    - 2월 10일 ~ 3월 9일 = 총 28일 (2월이 28일인 경우 가정)
    - 2월 26일 ~ 3월 9일 = 12일 남음
    - 즉, 남은 비율 = 12 / 28 = 약 42.86%
2. 가격 차이 계산
    - Premium - Pro 차액: $60 - $20 = $40 차이
    - 이 $40를 남은 기간만큼만 받으면 됨 : $40 × (12 / 28) = $17.14
    - 추가 결제 금액 ≈ $17.14
3. 크레딧 추가 지급량 계산 
    - 월 전체 크레딧 차이: 60,000 - 20,000 = 40,000 차이
    - 남은 기간만큼 제공: 40,000 × (12 / 28) = 17,143 크레딧
    - 추가 지급 크레딧 ≈ 17,143

### 동일 서비스 등급, 월간 → 연간 업그레이드
- 결제 페이지로 이동 (차액 결제 발생)
- 동일한 등급, 월 → 연 기간 업그레이드
- 즉시 반영
- 연간 요금에 남은 월간 요금 차감 적용
- 즉시 20,000 크레딧 제공(결제일 리셋)
- 결제일 리셋, 새로운 결제일 적용(다음년 자동결제)

[예상 시나리오]
- 현재 요금제: Pro
    - $20 / 월
    - 매월 20,000 크레딧
- 변경 요금제: Pro
    - $200 / 연
    - 매월 20,000 크레딧
- 결제 주기 시작일: 매월 10일
- 오늘 날짜: 2월 26일
(현재 사용 구간: 2월 10일 ~ 3월 9일, 남은 기간: 2월 26일 ~ 3월 9일 (남은 일:12일))
1. 남은 월간 요금 환불 계산
    - 남은 비율: 12 / 28 = 42.86%
    - 남은 금액: $20 × 42.86% = $8.57
    - $8.57 금액 차감
2. 연간 요금 계산
    - 연간 요금: $200
    - 기존 월간 남은 금액 차감: $200 - $8.57 = $191.43
    - 최종 추가 결제 금액:  $191.43 결제
3. 크레딧 지급 방식
    - 바로 20,000 크레딧 제공
    - 매월 20,000씩 크레딧 제공
4. 다음 결제일 
    - 2027년 2월 26일

### 서비스 등급 및 월간 → 연간 업그레이드
- 결제 페이지로 이동 (차액 결제 발생)
- 등급 및 월 → 연 기간 업그레이드
- 즉시 반영
- 연간 요금에 남은 월간 요금 차감 적용
- 즉시 60,000 크레딧 제공(결제일 리셋)
- 결제일 리셋, 새로운 결제일 적용(다음년 자동결제)
[예상 시나리오]
- 현재 요금제: Pro
    - $20 / 월
    - 매월 20,000 크레딧
- 변경 요금제: Premium
    - $600 / 연
    - 매월 60,000 크레딧
- 결제 주기 시작일: 매월 10일
- 오늘 날짜: 2월 26일
(현재 사용 구간: 2월 10일 ~ 3월 9일, 남은 기간: 2월 26일 ~ 3월 9일 (남은 일: 12일))
1. 기존 월간 요금 환불 계산
    - 남은 비율: 12 / 28 = 42.86%
    - 남은 금액: $20 × 42.86% = $8.57
    - 기존 월간 요금에서 $8.57 차감
2. 연간 Premium 요금 적용
    - 연간 요금: $600
    - 기존 남은 금액 차감: $600 - $8.57 = $591.43
    - 최종 추가 결제 금액: $591.43 결제
3. 바로 20,000 크레딧 제공
    - 매월 20,000씩 크레딧 제공
4. 다음 결제일 
    - 2027년 2월 26일

### 연간, 서비스 등급 업그레이드
- 결제 페이지로 이동 (차액 결제 발생)
- 연간 서비스 등급 업그레이드
- 즉시 반영
- 연간 차액 계산 후 남은 기간만큼만 청구
- 즉시 60,000 크레딧 제공(크레딧 계산 없이 업그레이드되는 등급 제공량 즉시 반영)
- 다음 년도 동일 일자 자동결제
[예상 시나리오]
- 현재 요금제: Pro
    - $200 / 연
    - 매월 20,000 크레딧
- 변경 요금제: Premium
    - $600 / 연
    - 매월 60,000 크레딧
- 오늘 날짜: 5월 20일
- 다음 결제일: 2027년 2월 26일
1. 남은 기간 계산
    - 현재 계약 구간은: 2026년 2월 26일 ~ 2027년 2월 26일 (1년)
    - 오늘은 2026년 5월 20일이므로
    - 경과 기간: 2/26 ~ 5/20 ≈ 83일 경과
    - 전체 기간 365일 기준
    - 남은 기간: 365 - 83 = 282일
    - 남은 비율:  282 / 365 ≈ 77.26%
2. 연간 차액 계산
    - 연간 요금 차이: $600 - $200 = $400 차이
    - 남은 기간만큼만 청구: $400 × 77.26% = $309.04
    - 최종 추가 결제 금액 : 약 $309.04 결제
    (실제 운영에서는 반올림하여 $309.04 또는 $309.05 처리)
3. 크레딧 지급 계산
    - Premium 크레딧 구조 즉시 적용
    - 60,000 크레딧 즉시 제공
    - 이후 매월 60,000 리셋
4. 다음 결제일 
    - 2027년 2월 26일

### 월간, 등급 다운그레이드
- 다운그레이드 페이지로 이동(환불 결제 없음)
- 월간 서비스 등급 다운그레이드
- 이번 달은 현재 등급 유지
- 새로운 등급은 다음 결제일부터 적용
- 다음 예약 결제

### 등급 동일, 연간 →  월간 다운그레이드
- 다운그레이드 페이지로 이동 내용 확인(구독 취소에 대한 환불 처리) → 결제 페이지로 이동(새로운 결제 진행)
- 현재 연간 구독 취소, 이번달은 기존 등급 사용 유지 → 새로운 월간 구독으로 다음달 부터 진행
- 정책 설명, 취소 시 할인 비용은 정상 월 비용으로 계산되어, 차액 환불되고, 새로은 플랜 결제 진행

### 서비스 등급 및 연간 → 월간 다운그레이드
- 다운그레이드 페이지로 이동 내용 확인(구독 취소에 대한 환불 처리) → 결제 페이지로 이동(새로운 결제 진행)
- 현재 연간 구독 취소, 이번달은 기존 등급 사용 유지 → 새로운 등급 및 월간 구독으로 다음달 부터 진행
- 정책 설명, 취소 시 할인 비용은 정상 월 비용으로 계산되어, 차액 환불되고, 새로은 플랜 결제 진행

# 추가 좌석 자동 결제 및 환불정책

## 정책사항

- 기간은 현재 상요중인 요금제 일정 따름
- 추가 좌석당 5$ 금액 청구 (한국 10%부가세 있음)
- 추가 시 즉시 적용
- 서비스 플랜 월 결제일에 기준에 따라 남은 일수 만큼 계산되어 결제 진행
    - 서비스 플랜 일정: 매월 10일 결제
    - 오늘 2/26 (2월 한달 28일, 2월 26일 ~ 3월 9일 = 12일 남음
    - 2월 26일 ~ 3월 9일 = 12일 남음
    - 즉, 남은 비율 = 12 / 28 = 약 42.86%
    - $ 5 X 42.86% = 2.143 결제
    - 다음달 3월 10일 $5 자동 결제(예약)
- 취소는  다음 결제일부터 적용 (예약 취소)
- 좌석 추가는 팀(team) 테넌트에만 해당됨
- 서비스 플랜 구독 취소 시 다음달 함께 자동 취소 적용
- 서비스 플랜이 다운그레이드 되서 팀(team)이 아닌  개인(personal)로 전환되었을때 다음달 함께 자동 취소 적용
- 서비스 플랜이 다운그레이되어 business → premium으로 전환했는데, 추가 좌석이 10명인데, premium은 추가로 확장할 수 있는 최대 좌석이 4명이기 때문에 6명의 추가 좌석은 다음달 변경 시점에 자동 취소
- 환불 없음


# 추가 충전 크레딧에 대한 결제 및 환불 정책

## 정책사항

- 추가 충전 크레딧은 선불 구매
- 기본 크레딧 소진 시 추가 충전 크레딧 사용 가능
- 구매 즉시 충전
- 추가 충전 크레딧은 환불되지 않음
- 사용기한은 구매일로부터 3년
