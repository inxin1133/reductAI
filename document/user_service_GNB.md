# 사용자 서비스 GNB (필수 메뉴 제안)

아래 메뉴는 사용자/테넌트 운영에 필요한 최소 구성안입니다.  
`users`, `tenants`, `pricing`, `billing`, `credits` 기반으로 설정·운영 흐름을 우선 배치했습니다.

## 일반
- 대시보드  
  내 테넌트의 사용량/결제/크레딧 현황을 한눈에 확인합니다.

## 계정/조직 (Users & Tenants)
- 내 프로필  
  사용자 기본 정보와 계정 설정을 관리합니다.
- 테넌트 정보  
  워크스페이스(테넌트) 기본 정보/상태를 확인·변경합니다.
- 멤버 관리  
  멤버 초대/권한/상태를 관리하고 팀 규모를 운영합니다.
- 역할/권한  
  팀 내 역할과 권한을 정의해 접근 범위를 제어합니다.

## 요금/가격 (Pricing)
- 요금제 안내  
  사용 가능한 요금제와 기능/한도를 비교합니다.
- 사용자 공개 요금표  
  모델/모달리티별 평균 요율을 확인합니다.

## LLM 사용/로그
- 모델 사용 로그  
  요청/응답, 상태, 토큰/비용 요약을 확인합니다.
- 모달리티별 사용량  
  이미지/오디오/비디오/검색 등 세부 사용량을 조회합니다.
- 비용 분석  
  기간/모델/프로젝트 기준 비용을 분석합니다.

## 구독/결제 (Billing)
- 구독 관리  
  현재 구독 상태, 업/다운그레이드, 갱신일을 관리합니다.
- 청구서  
  청구서 목록/상세와 과금 기간을 확인합니다.
- 청구 정보  
  결제 계정(청구 이메일/세금 정보 등)을 관리합니다.
- 결제 수단  
  카드/결제수단 등록 및 기본 결제수단을 관리합니다.
- 결제 내역  
  결제/환불 거래를 조회합니다.

## 크레딧 (Credits)
- 크레딧 잔액/계정  
  테넌트/사용자 크레딧 풀과 잔액을 확인합니다.
- 크레딧 충전  
  충전 상품 선택 및 결제를 진행합니다.
- 크레딧 사용 내역  
  사용/조정/만료 등 크레딧 변동 이력을 확인합니다.
- 크레딧 분배/이전  
  팀/사용자 간 크레딧 분배 및 회수 내역을 관리합니다.
- 크레딧 사용 분배  
  사용 이벤트별로 어떤 계정에서 차감됐는지 확인합니다.
- 사용 순서 설정  
  여러 계정 보유 시 소비 우선순위를 설정합니다.
